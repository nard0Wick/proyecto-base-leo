Indicaciones:

Tras haber realizado un fork del repositorio y disponer de una copia local,
crear una nueva rama y escribir las lineas de código necesarias 
para implementar una función que calcule la raíz x de un número y, 
posteriormente preparar los archivos y realizar un commit. 

Una vez se haya generado el commit, subir los cambios al repositorio en 
github será el siguiente paso, de manera que github terminará por resaltar una
diferencia de versiones entre ambas ramas (main/master y la anteriormente creada),
suscitando un pull request.
